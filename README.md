# About The Project
This is a small framework created for the purpose of studying networks.

# Required
* **Python Version 3**

# Supported environment
* **Kali Linux**
* **Parrot OS** 
* **Ubuntu**

# Installation:

Download and enter the Setup directory
```
git clone https://github.com/Endo65/Netools.git
cd Netools/Setup/
```
Execute the installer 
```
python install.py
```
Run netools.py
```
python netools.py
```
